## Getting Started

#### Front-End:

1. Clone the repo
 
 `git@github.com:Helene-Normant/PC-CastorlamaFront2.git`
 
2. Install npm packages

`npm install`
  
3. Run the project
  
`npm start run`

#### Back-End: 

1.Clone the repo 

`git@github.com:adatechschool/PC-CastorlamaBack.git`

2. Express

`npm install`

3. Run the project  

`npm start run`
